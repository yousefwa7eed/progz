تعليمات رفع الموقع عالمياً
=============================

الملفات في هذا المجلد:
- index.html          ← صفحة التحميل
- logo.png            ← شعار المؤسسة
- AlmajidApp.zip      ← ملف التطبيق (47 MB)

طريقة النشر (Netlify - الأسهل):
1. افتح المتصفح واذهب إلى: https://app.netlify.com/drop
2. اسحب مجلد "site" بالكامل وأفلته في الصفحة
3. بعد ثوانٍ، سيظهر رابط الموقع (مثل: https://abc123.netlify.app)
4. أعطِ هذا الرابط للمشتري لتقييم التطبيق

طريقة النشر (GitHub Pages):
1. أنشئ مستودع جديد على GitHub
2. ارفع ملفات المجلد site إلى المستودع
3. اذهب إلى Settings → Pages واختر main branch
4. سيظهر الرابط بعد دقيقة

طريقة النشر (Vercel):
1. اذهب إلى: https://vercel.com/new
2. اسحب مجلد site بالكامل
3. سيظهر الرابط فوراً

ملاحظة: الملفات جاهزة تماماً. كل ما عليك هو سحب مجلد site إلى إحدى الخدمات أعلاه.
