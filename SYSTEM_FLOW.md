# مخطط سير العمليات - نظام الماجد
## Almajid System Flow Diagram

---

## 1. سير عملية تقديم المساعدة (الحالة)

```
المستفيد → استقبال الحالة → دراسة الحالة → اعتماد → صرف → متابعة
```

### تفصيل المراحل:

```
┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐    ┌────────────┐
│  استقبال   │ → │   دراسة    │ → │  اعتماد    │ → │   صرف     │ → │  متابعة   │
│  الحالة    │    │  الحالة    │    │  الحالة    │    │ المساعدة   │    │  وتقييم   │
└────────────┘    └────────────┘    └────────────┘    └────────────┘    └────────────┘
       │                │                │                │                │
       ▼                ▼                ▼                ▼                ▼
  تسجيل بيانات     بحث اجتماعي      مدير المشاريع    صرف المساعدة     إعادة تقييم
  المستفيد +       تقييم الاحتياج   أو المدير         طباعة إيصال      كل 3-6 شهور
  إرفاق            درجة الأولوية    التنفيذي          تحديث الحالة     إغلاق أو تجديد
  المستندات        اقتراح المساعدة  يعتمد أو يرفض                        أو تحويل
```

### حالات الحالة (Case Statuses):
```
جديد → قيد_الدراسة → معتمد → قيد_الصرف → منصرف → للمتابعة → مغلق
                                          ↓
                                       مرفوض
```

## 2. سير عملية التبرع

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  وصول تبرع   │ → │  تسجيل التبرع │ → │  إصدار إيصال  │ → │  تحديث سجل   │
│  (نقدي/عيني) │    │  في النظام    │    │  رسمي         │    │  المتبرع     │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
       │                                                           │
       ▼                                                           ▼
  تصنيف التبرع                                                إرسال إشعار
  نقدي ← إيداع بنكي                                           شكر للمتبرع
  عيني ← إلى المخزون                                          تقرير أثر
  كفالة ← تخصيص لمستفيد
  مشروع ← تخصيص لمشروع
  زكاة ← وسم كزكاة
  صدقة ← وسم كصدقة
```

## 4. سير عملية المشروع الخيري

```
┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐   ┌────────┐
│ فكرة   │ → │ دراسة  │ → │ اعتماد │ → │ تنفيذ  │ → │ تقييم  │ → │ إغلاق  │
│ مشروع  │   │ جدوى   │    │ ميزانية│    │        │    │ نتائج  │    │        │
└────────┘   └────────┘   └────────┘   └────────┘   └────────┘   └────────┘
                  │            │
                  ▼            ▼
             خطة مالية     موافقة المدير
             تفصيلية       التنفيذي
```

### أنواع المشاريع:
| المشروع | الوصف |
|---------|-------|
| توزيع غذاء | سلال رمضانية - مواد جافة - وجبات |
| كسوة الشتاء | ملابس - بطانيات - مدافئ |
| سقيا المياه | آبار - شبكات مياه - صهاريج |
| دعم العلاج | عمليات - أدوية - علاج |
| ترميم المنازل | إصلاح - بناء - صيانة |
| مساعدات تعليمية | أجهزة - كتب - رسوم دراسية |
| مساعدات موسمية | عيد الأضحى - المولد - العودة للمدارس |
| إفطار صائم | وجبات إفطار رمضانية |

## 10. سير الموافقات (Approval Workflow)

```
مساعدات نقدية ≥ 5000 ريال:
موظف ← مشرف ← مدير مالي ← مدير تنفيذي

مساعدات نقدية < 5000 ريال:
موظف ← مشرف ← مدير مالي

مساعدات عينية:
موظف ← مسؤول مخازن ← مدير مشاريع

مشاريع جديدة:
مدير مشاريع ← مدير مالي ← مدير تنفيذي ← مجلس إدارة (للمشاريع الكبيرة)

صرف مرتبات:
موظف موارد ← مدير مالي ← مدير تنفيذي
```
